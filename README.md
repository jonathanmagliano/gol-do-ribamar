## Compatibilidade

- Mac
- Linux
- Windows

## Instalação
Instalação global (pasta):

    npm install -g

Execução via CLI:
    
    goldoribamar

## Rodando com [npx](https://www.npmjs.com/package/npx)
Rode o comando:

    npx goldoribamar
